Quantitative Macro Models

Over the years I have written a collection of code implementations for key macroeconomic models as part of my learning and research process. I hope they prove useful to others interested in quantitative approaches to long-run development.

Most of the models are written in Python using Numba, and include heterogeneous agents, either on the household or firm side. The collection continues to grow, and you can find multiple computational approaches for each model.
